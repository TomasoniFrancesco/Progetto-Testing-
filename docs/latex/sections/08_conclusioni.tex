\section{Conclusioni}

Il progetto copre le quattro sezioni richieste: requisiti, modello Asmeta (con
scenari corretti/nuovi e model checking), contratti JML sul nucleo di dominio, e
implementazione Java con test (JUnit, MC/DC, Mockito, Model Based Testing,
combinatorial testing), UI Vaadin, CI e analisi statica.
